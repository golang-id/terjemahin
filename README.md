terjemahin
=========

Utilitas CLI untuk menterjemahkan teks (menggunakan Yandex API)

## Install

```
go get github.com/golang-id/terjemahin
```

## Usage

```
$ terjemahin "This works like a charm"
Ini bekerja seperti mantra

$ terjemahin -from=id -to=fr "selamat makan"

```
